# LR(1)语法分析器测试用例

## 测试用例1: 经典算术表达式文法
### 文法:
E -> E + T | T
T -> T * F | F
F -> ( E ) | id

### 开始符号: E

### 测试输入串:
1. id + id * id
2. ( id + id ) * id
3. id * ( id + id )
4. ( ( id ) )
5. id + id + id
6. id * id * id

### 预期结果: 全部成功

---

## 测试用例2: 简单赋值语句文法
### 文法:
S -> id = E
E -> E + T | T
T -> T * F | F
F -> ( E ) | id | num

### 开始符号: S

### 测试输入串:
1. id = id + num
2. id = ( id + num ) * id
3. id = id

### 预期结果: 全部成功

---

## 测试用例3: 包含空产生式的文法
### 文法:
S -> A B
A -> a | ε
B -> b

### 开始符号: S

### 测试输入串:
1. a b
2. b

### 预期结果: 全部成功

---

## 测试用例4: LR(1)特有文法（SLR(1)无法处理）
### 文法:
S -> L = R | R
L -> * R | id
R -> L

### 开始符号: S

### 测试输入串:
1. id = id
2. * id = id
3. id

### 预期结果: LR(1)成功，SLR(1)可能有冲突

---

## 测试用例5: 条件语句文法
### 文法:

### 开始符号: S

### 测试输入串:

3. if id then if id then id else id

### 预期结果: 全部成功

---

## 测试用例6: 错误输入测试
### 文法:

## 测试用例7: 复杂嵌套表达式

### 开始符号: S

## 使用说明

1. **基础测试**: 使用测试用例1-3验证基本功能
2. **LR(1)特性测试**: 使用测试用例4验证LR(1)的优势
3. **复杂语法测试**: 使用测试用例5-9验证复杂语法处理
4. **错误处理测试**: 使用测试用例6验证错误处理
5. **边界测试**: 使用测试用例10验证边界情况
6. **性能测试**: 使用性能测试用例验证大型文法处理

## 测试步骤

1. 在"文法输入"标签页输入测试文法
2. 设置正确的开始符号
3. 点击"解析文法"
4. 点击"构建分析器"
5. 在"LR(1)分析表"标签页检查是否有冲突
6. 在"语法分析过程"标签页输入测试串
7. 点击"开始分析"查看结果
8. 验证结果是否符合预期

## 预期行为

### 成功情况:
- 分析步骤完整显示
- 最后一步显示"接受"
- 结果状态显示"✓ 分析成功"

### 失败情况:
- 分析步骤显示到错误位置
- 最后一步显示错误信息
- 结果状态显示"✗ 分析失败"
- 详细信息中包含错误描述

### 冲突情况:
- 构建分析器时显示警告
- 冲突信息标签页显示具体冲突
- 可能影响分析结果的准确性
