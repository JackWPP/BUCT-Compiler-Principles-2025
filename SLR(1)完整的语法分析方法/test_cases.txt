# SLR(1)语法分析器测试用例

## 测试用例1: 经典算术表达式文法
E -> E + T | T
T -> T * F | F
F -> ( E ) | id

开始符号: E

测试字符串:
- id (应该成功)
- id + id (应该成功)
- id * id (应该成功)
- id + id * id (应该成功，优先级：* > +)
- ( id + id ) * id (应该成功)
- id + (应该失败，语法错误)
- ) id ( (应该失败，语法错误)

## 测试用例2: 简单递归文法
S -> a S b | ε

开始符号: S

测试字符串:
- ε (空串，应该成功)
- a b (应该成功)
- a a b b (应该成功)
- a a a b b b (应该成功)
- a b b (应该失败，不匹配)
- a a b (应该失败，不匹配)

## 测试用例3: 有冲突的文法（测试冲突检测）
S -> E
E -> E + E | E * E | id

开始符号: S

这个文法应该有移进-归约冲突，因为在遇到 + 或 * 时，
不知道是移进还是归约。程序应该检测并报告这些冲突。

## 测试用例4: 括号匹配文法
S -> ( S ) S | ε

开始符号: S

测试字符串:
- ε (空串，应该成功)
- ( ) (应该成功)
- ( ( ) ) (应该成功)
- ( ) ( ) (应该成功)
- ( ( ) ) ( ) (应该成功)
- ( ( (应该失败，括号不匹配)
- ) ) (应该失败，括号不匹配)

## 测试用例5: 简单赋值语句文法
S -> id = E
E -> E + T | T
T -> id | num

开始符号: S

测试字符串:
- id = id (应该成功)
- id = num (应该成功)
- id = id + num (应该成功)
- id = id + id + num (应该成功)
- = id (应该失败，缺少左值)
- id = (应该失败，缺少右值)

## 使用说明
1. 在GUI程序的"文法输入"标签页中输入上述文法
2. 设置相应的开始符号
3. 点击"解析文法"和"构建分析器"
4. 在"语法分析过程"标签页中输入测试字符串
5. 观察分析结果和详细步骤
